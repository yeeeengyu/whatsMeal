# WhatsMeal

경북소프트웨어마이스터고등학교 급식을 작업표시줄에서 바로 확인할 수 있는 작은 급식 위젯입니다.

앱을 실행하면 Windows 작업표시줄 트레이에 아이콘이 생기고, 아이콘을 클릭하면 오늘의 점심과 저녁 급식을 바로 볼 수 있습니다.

## 다운로드

GitHub Releases에서 최신 `WhatsMeal.exe` 파일을 다운로드하세요.

다운로드 후 원하는 위치에 두고 실행하면 됩니다. 설치 과정은 필요 없습니다.

## 사용 방법

1. `WhatsMeal.exe`를 실행합니다.
2. 작업표시줄 오른쪽 트레이 영역에 급식 아이콘이 표시됩니다.
3. 아이콘을 클릭하면 급식 위젯이 열립니다.
4. 오늘 급식이 자동으로 표시됩니다.

트레이 아이콘이 바로 보이지 않으면 작업표시줄의 숨겨진 아이콘 영역을 확인해 주세요.

## 주요 기능

- 오늘 급식 바로 보기
- 점심과 저녁 메뉴 확인
- 날짜를 선택해서 다른 날 급식 조회
- 급식 내용 복사
- 새로고침
- 트레이 아이콘으로 간단히 열고 닫기

## 실행이 느릴 때

처음 실행할 때 Windows 보안 검사가 진행되면서 잠시 늦게 열릴 수 있습니다. 조금 기다린 뒤 작업표시줄 트레이 영역을 확인해 주세요.

## 문제가 있을 때

앱이 실행되지 않거나 급식이 보이지 않으면 다음을 확인해 주세요.

- 인터넷 연결이 되어 있는지 확인
- 작업표시줄의 숨겨진 아이콘 영역 확인
- 앱을 종료한 뒤 다시 실행

문제가 계속되면 아래 로그 파일을 함께 전달해 주세요.

```text
%APPDATA%\WhatsMeal\client.log
```

## 지원 환경

- Windows 10 이상
- Windows 11

Linux와 macOS용 앱은 아직 제공하지 않습니다.
